sprime [![GoDoc](https://godoc.org/github.com/otiai10/sprime?status.svg)](https://godoc.org/github.com/otiai10/sprime)
==========

- Find primary numbers
```sh
% sprime p 20
[2 3 5 7 11 13 17 19]
```
- Factorize numbers
```sh
% sprime f 329
[7 47]
```
- Reduce fractions
```sh
% sprime r 144/360
2/5
```
